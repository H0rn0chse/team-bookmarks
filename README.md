## Project setup

1. Installing dependencies
```
npm install
```
2. Install Pre-commit hook
```
npm run copy-pre-commit
```
3. Start dev setup (hot-reloads)
```
npm run dev
```
4. Build for production (automatically done by pre-commit hook)
```
npm run build
```
